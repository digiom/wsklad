== Description ==
Implementation of a mechanism for flexible exchange of various data between Moy Sklad
and a site running WordPress using the WooCommerce plugin.

== Translations ==
* English - default, always included
* Russian - always included

== Installation ==
Archive extract and upload folder "wsklad" to /wp-content/plugins (final path: /wp-content/plugins/wsklad).
